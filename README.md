## Super Heroes App

App hecha en React usando Vite, como dependencias:

- Ant Design
- Styled Components
- React Hook Form
- Yup
- Redux Toolkit

Consumo de la API `https://superheroapi.com`

Un pequeño server para montar el build de produción, con:

- Express
- Dotenv
- Cors
- Compression

Deploy: `https://jstatik-super-heroes.netlify.app`
